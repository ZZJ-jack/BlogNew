## 技术分享

### Vue 3 新特性

Vue 3 带来了许多新特性，包括：

- Composition API
- Teleport
- Fragments
- 更好的 TypeScript 支持

```vue
<template>
  <div>
    <h1>{{ title }}</h1>
  </div>
</template>

<script setup>
import { ref } from 'vue';

const title = ref('Hello Vue 3');
</script>
```

> Vue 3 是一个非常强大的前端框架
